Nessa frase temos um termo em negrito usando a tag <b>(não semanticao/bsoleta)

Nesta frase, tenho um termo em destaque usando a tag <strong> (semantica)

Itálico/Ênfase

Nesta frase, temos um termo em ítalico usando a tag <i> (não semantica/obsoleta)

Nesta frase, temos um termo em ênfase usando a tag <em>(semantica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark> desta cor.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado, isso usando <del>

Texto inserido

Podemos marcar um texto como inserido para dar um ênfase e indicar que ele foi adicionado depois, isso usando a tag <ins>

Texto sobrescrito

Para inserir coisas do tipo x20+3, isso com a tag <sup>

Texto subescrito

Para inseiri coisas do tipo H2O, isso com a tag <sub>
